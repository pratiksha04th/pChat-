# PChat – Flutter AI Powered Chat Application

PChat is a **real-time chat application built with Flutter and Firebase**, enhanced with **AI-powered conversations**.
It supports **private messaging, group chats, push notifications, and intelligent AI interaction** for a modern chat experience.

---

## 🚀 Features

* **Firebase Authentication**

  * Email & Password login
  * Email verification

* **Real-time Chat**

  * One-to-one messaging
  * Instant message updates using Firebase Realtime Database

* **Group Chats**

  * Create chat groups
  * Multiple participants

* **Push Notifications**

  * Firebase Cloud Messaging (FCM)
  * Unread message counters

* **AI Integration**

  * AI-powered chat interaction
  * Real-time conversational AI support

* **Message Management**

  * Delete messages
  * Message selection

* **Search**

  * Search users
  * Search groups

* **Optimized Chat Experience**

  * Unread message counters
  * Message sorting
  * Real-time updates

---

## 🛠 Tech Stack

| Technology                 | Usage                       |
| -------------------------- | --------------------------- |
| Flutter                    | Mobile UI framework         |
| Firebase Authentication    | User authentication         |
| Firebase Realtime Database | Real-time chat data         |
| Firebase Cloud Messaging   | Push notifications          |
| GetX                       | State management            |
| AI API (Hume AI)           | AI-powered chat interaction |
